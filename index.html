1. Subsystem

  • Class: UUnicornBallisticsSubsystem (World Subsystem, tickable)
  • Blueprint Access: “Get Unicorn Ballistics Subsystem” → singleton instance
  • Net Execution: EUBSimTaskNetMode on configuration (None, Client, Server, Both); no automatic replication: user-managed via RPCs/multicasts for multiplayer
  • Thread Safety: Most functions thread-safe; handles adding/removing/firing bullets/projectiles
  • Purpose: Central hub for simulation; exists based on SimulationNetMode (server/client/both)

4. Data Assets

  • UBulletDataAsset - one per caliber (Mass g, Diameter mm, BallisticCoefficient, DragModel EUBGModel, Variation EUBBulletVariation, Penetration multipliers, Trail/Impact VFX/SFX, Default Spread)
  • UProjectileDataAsset - similar to UBulletDataAsset but for general projectiles (e.g., rockets, arrows)
  • PhysicalMaterial - extra “Unicorn Ballistics” category with full FUBPhysMatProperties; mapped via tags/surface types

11. Math Models (for physics reference).

Other document (Math/Physics Document) is more efficient for the math/physics models and explanation.

  • Motion: 3DOF with gravity g=-9.81 m/s² and drag; Δx/Δt = v(t), Δv/Δt = [0, 0, -9.81] - F_D(v)/m
  • Drag: F_D(s) = 1/2 C_D A ρ s² (ρ=1.225 kg/m³)
  • Integration: Velocity-Verlet for accuracy; x_{n+1} = x_n + v(t_n) Δt + 1/2 \dot{v}(t_n) Δt², v_{n+1} = v_n + 1/2 [\dot{v}(t_n) + \dot{v}(t_n + Δt)] Δt
  • Penetration: Based on material properties (energy loss J/cm, resistance)
  • Ricochet: Angle cutoff, probability
